# pppicker
 
- 功能: lch 拾色器

- 依赖库:
  - vanjs - 最轻量级的前端响应式框架
  - culori - 色彩转换库
  - tauri - desktop 端框架

- 核心代码: ./src/main.ts

- 未来优化方向:
  - 增加横截面色盘, 目前这个只有纵截面
  - 增加更多色彩模型, 比如 oklch, 谷歌的 hct, 进而增加更多色彩系统之间的转换
  - 让这个拾色器可以直接控制绘画软件中的颜色, 可能还要在绘画软件中写一个专门的插件(强烈吐槽 ps 的 uxp 插件系统🤮).

- 补充: 有一个函数是用 ai 写的, 猜猜是哪个😋
